# Babelbeez Voice Agent – Google Tag Manager Template

This repository contains the Google Tag Manager tag template for embedding the **Babelbeez Voice Agent** on any website.

## Usage

1. Import `template.tpl` into Google Tag Manager as a custom tag template, or install it from the Community Template Gallery.
2. Create a new tag using the **Babelbeez Voice Agent** template.
3. In the tag configuration, set:
   - **Public Chatbot ID** – the ID from your Babelbeez dashboard.
4. Choose a trigger (e.g. `All Pages`) and publish your container.

## Files

- `template.tpl` – exported GTM template.
- `metadata.yaml` – gallery metadata and version history.
- `LICENSE` – Apache 2.0 license.
